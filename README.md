# Google Maps – Saved Places Redesign

## Overview
A UX case study exploring how Google Maps' Saved Places feature can better support trip planning, organization, and collaboration.

## Problem
Saved Places become cluttered as users save more locations, making retrieval and planning difficult.

## Solution
- Smart Collections
- Collaborative Itineraries
- Context-aware categorization
- Automatic organization

## Research
- Travelers
- Students
- Commuters
- Delivery workers

## Design Process
- Problem Definition
- User Research
- Personas
- User Flows
- Wireframes
- Product Recommendations

## Outcome
The redesign reduces manual organization, improves retrieval, and supports collaborative planning.
